# react-components-demo3

## What is this?
A beginner React project that explains component-based architecture
using Create React App, TypeScript, and Rspack.

## How React Works
1. Browser loads index.html
2. Rspack loads bundle.js
3. React mounts into #root
4. App component loads child components

## How to Run the Project

1. Install dependencies (only once):
   npm install

2. Start the development server:
   npm start

3. Open in browser:
   http://localhost:3000
