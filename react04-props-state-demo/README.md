# React Props and State Demo (Beginner)

## What this project teaches
- What React is
- What components are
- What props are
- What state is
- How React updates UI automatically

## How to run

1. Install dependencies
   npm install

2. Start dev server
   npm start

3. Open browser
   http://localhost:3000

## Folder structure

public/
  index.html   → HTML shell
src/
  index.tsx    → React entry
  App.tsx      → Props and State example
rspack.config.js → Bundler config
tsconfig.json → TypeScript config
