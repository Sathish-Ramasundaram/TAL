# react-jsx-demo2

## What is this project?
A beginner-friendly React project to demonstrate JSX syntax and dynamic content.

## How to Run
1. Install dependencies:
   npm install

2. Start the server:
   npm start

3. Open in browser:
   http://localhost:3000

## What You Learn
- JSX syntax vs HTML
- Embedding JavaScript inside JSX
- Using dynamic variables
- Rendering lists
- Function components
