# TAL — Learning Demos

This repository collects several small demos on how the DOM and React work.

**Projects included**
- **normal-dom-demo**: Plain HTML + JavaScript demonstration of real 

- **react01-components-demo**: Small React app (create-react-app scripts) demonstrating components 

- **react02-jsx-demo**: JSX-focused React examples using `react-scripts` 

- **react03-virtual-dom-demo**: Demonstrates React's virtual DOM concepts (uses `react-scripts`) 

- **react04-props-state-demo**: Examples of props & state; uses `rspack` for development 

**Quick start (per demo)**
- normal-dom-demo: open the file [normal-dom-demo/index.html](normal-dom-demo/index.html) in a browser (no install needed).
- react01-components-demo / react02-jsx-demo / react03-virtual-dom-demo:
  - Install dependencies and start the dev server:

```bash
cd react01-components-demo
npm install
npm start
```

Replace the folder name for the other demos (`react02-jsx-demo`, `react03-virtual-dom-demo`).

- react04-props-state-demo (rspack):

```bash
cd react04-props-state-demo
npm install
npm start
```

Notes: some demos use `react-scripts` (CRA) while `react04-props-state-demo` uses `rspack serve` for development.

**Prerequisites**
- Node.js and npm installed (Node 16+ recommended).
