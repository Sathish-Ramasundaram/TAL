
## Description
This project demonstrates handling user interactions in React using event handlers.
It is bundled and served using Rspack.

## Tech Stack
- React
- TypeScript
- Rspack

## Scripts
npm start  -> start dev server
npm run build -> production build

## Concepts Covered
- Button click events
- Input change events
- Form submission
- React Synthetic Events
- useState

## Learning Outcome
- Understand event handling in React
- Understand how Rspack bundles React apps
